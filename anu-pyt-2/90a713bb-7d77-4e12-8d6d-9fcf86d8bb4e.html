Travel Date Weekday Finder


Liam and Emma are developing a travel itinerary system that needs to identify the day of the week for specific travel dates. Each date is given in the format YYYY-MM-DD, and they must determine which weekday it falls on, such as Monday, Tuesday, etc.


Your task is to help Liam and Emma write a program that finds the day of the week for a given travel date.




Input Format


    

  • A single line containing a string date in the format YYYY-MM-DD.
    • 





Output Format


    

  • Print the day of the week (e.g., Monday, Tuesday, etc.) corresponding to the given date.
    • 





Constraints


    

  • The year range is between 1971 and 2100.
    • 





Example 1


Input


2025-11-06


Output


Thursday


Explanation


    

  • The date 2025-11-06 falls on a Thursday.

        self.product_price=new_price
