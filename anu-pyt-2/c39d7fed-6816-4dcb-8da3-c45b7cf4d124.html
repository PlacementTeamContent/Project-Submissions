Student Class


You are given an empty Student class. Your task is to implement this class by adding the required attributes and methods as described below.




**Features to be added to **_Student_** class**

- **Add below attributes**
 - The **Student** class should have the following attributes
   - student_name (string)
   - student_id (integer)
   - grade (string)
   - marks (integer)
 - Add a constructor to initialize above attributes 
- **Add below methods**
 - **update_grade()**: 
    - **Parameters**: string new_grade.
    - **Functionality**: Updates the student’s grade to new_grade.

 - **add_marks()**: 

     - **Parameters**: integer additional_marks.
     - **Functionality**: Increases the student’s marks by additional_marks.

 - **calculate_percentage()**: 
   - **Parameters**: integer total_marks.
   - **Functionality**: Calculates the percentage based on marks and total marks, and prints it in the format:


    Percentage:{percentage}%

class Student:
    def __init__(self,student_name,student_id,grade,marks):
       self.student_name=student_name
       self.student_id=int(student_id)
       self.grade=grade
       self.marks=int(marks)
    def update_grade(self,new_grade):
        self.grade=new_grade
    def add_marks(self,additional_marks):
        self.marks+=additional_marks
    def calculate_percentage(self,total_marks):
        percentage=int((self.marks/int(total_marks))*100)
        print(f"Percentage:{percentage}%")

Your task is to help Sophie and Ethan create a program that identifies the day of the week for a given historical event date.




Input Format
