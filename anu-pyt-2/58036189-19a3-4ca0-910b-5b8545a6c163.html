Event Day Detector


Ravi and Neha are designing an event reminder tool that highlights which day of the week each event occurs on. Every event date is provided in the format YYYY-MM-DD, and they need to find out whether it falls on Monday, Tuesday, Wednesday, etc.


Your task is to assist Ravi and Neha in writing a program that determines the day of the week for the given event date.




Input Format


    

  • A single line containing a string date in the format YYYY-MM-DD.
    • 





Output Format


    

  • Print the day of the week (e.g., Monday, Tuesday, etc.) corresponding to the given date.
    • 





Constraints


    

  • The year range is between 1971 and 2100.
    • 





Example 1


Input


2025-11-06


Output


Thursday


Explanation


    

  • The date 2025-11-06 falls on a Thursday.

- **Add below methods**

class BankAccount:
    def __init__ (self, account_holder, account_number, balance, bank_name):
        self.account_holder=account_holder
        self.account_number=account_number
        self.balance=balance
        self.bank_name=bank_name
    def deposit(self,amount):
        self.balance+=amount
    def withdraw(self, amount):
        self.balance-=amount
    def update_bank_name(self,new_bank_name ):
        self.bank_name=new_bank_name
